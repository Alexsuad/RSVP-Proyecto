# Carpeta de Imágenes para Emails y Páginas de Invitados

Esta carpeta contiene las imágenes utilizadas en:
- **Correos electrónicos HTML** (headers/banners de plantillas de email)
- **Páginas del frontend de invitados** (fondos, headers)

## Imágenes Requeridas

### 1. Header de Emails (`email-header.jpg` o `.png`)
- **Dimensiones sugeridas**: 800px ancho × 180-200px alto
- **Temática**: Boda (flores, anillos, motivos románticos)
- **Uso**: Banner superior de todos los correos (Guest Code, Confirmación, Recordatorio)

### 2. Background de Páginas de Invitados (opcional, `guest-page-bg.jpg`)
- **Dimensiones sugeridas**: 1920px ancho (se escalará automáticamente)
- **Temática**: Boda elegante, colores suaves
- **Uso**: Fondo de páginas como Login, RSVP Form, Confirmed

## Instrucciones

1. Sube las imágenes a esta carpeta (`frontend/public/assets/images/email/`)
2. Asegúrate de usar nombres descriptivos y en minúsculas (ej: `email-header.jpg`)
3. Notifica cuando las imágenes estén listas para que actualicemos las referencias en el código

## Notas Técnicas

- Las imágenes aquí serán accesibles públicamente vía URL: `https://tu-dominio.com/assets/images/email/nombre-imagen.jpg`
- Se recomienda optimizar las imágenes antes de subirlas (compresión sin pérdida de calidad)
- Formatos soportados: JPG, PNG, WebP
